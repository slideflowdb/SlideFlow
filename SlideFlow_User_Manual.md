## 1. Login Page

The login page is the entry point to SlideFlow. After the cursive intro animation, you are redirected here to authenticate.

![Login Page — Full View](docs/manual-screenshots/login_page_full_1774479579973.png)

---

### SlideFlow Brand Title

The animated cursive "SlideFlow" logo displayed above the login card. This is the branding for the application and is not interactive.

![SlideFlow Brand Title](docs/manual-screenshots/login_brand_title_zoomed_1774479620897.png)

---

### Username Field

Enter your assigned username here. The field includes a user icon indicator on the left side.

![Username Input](docs/manual-screenshots/login_username_input_1774479587156.png)

---

### Password Field

Enter your password here. The field includes a lock icon on the left and an eye/toggle icon on the right.

- **Eye Icon (👁)**: Click to toggle password visibility between hidden (dots) and visible (plain text).

![Password Input](docs/manual-screenshots/login_password_input_1774479588155.png)

---

### Remember Me Checkbox

Check this to save your login session so you don't have to re-enter credentials on your next visit.

![Remember Me](docs/manual-screenshots/login_remember_me_checkbox_1774479589450.png)

---

### Login Button

Click to submit your credentials and access the dashboard. The button displays "Signing in..." while authenticating. If credentials are invalid, a red error message appears above the button.

![Login Button](docs/manual-screenshots/login_button_1774479590455.png)

---

### Copyright Footer

Displays the copyright notice and branding attribution at the bottom of the page.

![Copyright](docs/manual-screenshots/login_copyright_text_zoomed_1774479625079.png)

## 4. Projects Page

The Projects page displays all your slide show projects in a card grid layout.

![Projects Page](docs/manual-screenshots/projects_page_full_1774479884042.png)

---

### Add Project Button

Click to open the "Create New Project" dialog. Located in the top-right corner of the page.

![Add Project](docs/manual-screenshots/add_project_button_1774479888687.png)

---

### Create New Project Dialog

A modal dialog that appears when you click "Add Project":

- **Project Name Field**: Enter a unique name for your new project. An error appears if a project with the same name already exists (case-insensitive).
- **Create & Edit Button**: Creates the project and immediately opens it in the slide editor.

![Create Project Dialog](docs/manual-screenshots/create_project_dialog_1774479895871.png)

---

### Project Card

Each project is displayed as a card showing:

- **Project Preview**: A thumbnail of the first slide
- **Project Name**: The name you gave the project
- **Slide Count**: How many slides are in the project
- **Timestamps**: When the project was created and last modified
- **Edit Button**: Click to open the project in the slide editor
- **Delete Button**: Remove the project permanently

![Project Card](docs/manual-screenshots/project_card_example_1774479889717.png)

## 7. Slide Editor

The editor is the core workspace where you design individual slides for your digital signage presentations.

![Editor — Full View](docs/manual-screenshots/editor_full_page_1774480046165.png)

---

### 7.1 Editor Toolbar

Located on the left side of the editor. These tools control what you can add or manipulate on the canvas.

---

#### Cursor / Select Tool

**Default tool.** Click to select, move, drag, and resize elements on the canvas. When active, clicking any element highlights it with resize handles.

![Cursor Tool](docs/manual-screenshots/cursor_tool_1774480047805.png)

---

#### Text Tool (T)

Adds a new text box to the current slide. After clicking, a text element is placed on the canvas that you can double-click to edit. Use the Properties panel to change font, size, color, alignment, and weight.

![Text Tool](docs/manual-screenshots/text_tool_1774480048810.png)

---

#### Upload / Image Tool

Opens the image upload dialog. You can either upload a new image from your computer or select an existing image from the Content Library. The image is placed on the canvas and can be resized.

![Image Tool](docs/manual-screenshots/image_tool_1774480049833.png)

---

#### Template Tool

Opens the Templates dialog showing a searchable gallery of pre-designed slide templates. Templates are organized by genre (Social Media, Events, Congratulations, Employee Recognition, Promotions). Click a template and then "Import Template" to apply it to your project.

![Template Tool](docs/manual-screenshots/template_tool_1774480050849.png)

---

#### Shapes Tool

Adds geometric shapes to the canvas. Available shapes include rectangles and circles. Shapes can be resized, repositioned, and styled with custom colors via the Properties panel.

![Shapes Tool](docs/manual-screenshots/shapes_tool_1774480058057.png)

---

#### Widget / Layout Tool

Adds pre-built widgets or complex layout grids to your slide. Useful for creating structured content like dashboards or multi-section layouts.

![Widget Tool](docs/manual-screenshots/widget_tool_1774480059064.png)

---

### 7.2 Actions Panel

Quick-action buttons for common editing operations, located below the toolbar on the left side.

![Actions Section](docs/manual-screenshots/actions_section_zoom_1774480104031.png)

---

#### Undo

Reverts the most recent change (moving, resizing, adding, or deleting elements). Keyboard shortcut: **Ctrl+Z**.

---

#### Redo

Re-applies a previously undone change. Keyboard shortcut: **Ctrl+Y**.

---

#### Delete / Trash

Permanently removes the currently selected element from the canvas. Select an element first, then click this button.

---

### 7.3 Slides Panel

Located on the left side, below the Actions. Manages the slide sequence for your project.

---

#### Add Slide (+)

Click the **+** icon next to the "Slides" heading to add a new blank slide to the end of your project.

![Add Slide](docs/manual-screenshots/slides_add_button_1774480060059.png)

---

#### Slide Thumbnails

Small previews of each slide. Click a thumbnail to navigate to that slide on the canvas.

---

#### Duplicate Button

Creates an exact copy of the currently selected slide, including all its elements and styling. Appears below the slide thumbnails.

![Duplicate](docs/manual-screenshots/duplicate_button_1774480061066.png)

---

#### Delete Slide Button

Permanently removes the currently selected slide from the project. Appears next to the Duplicate button.

![Delete Slide](docs/manual-screenshots/delete_button_1774480062072.png)

---

### 7.4 Top Bar Controls

The horizontal bar across the top of the editor provides project-level controls.

---

#### Project Name Field

Displays the current project name. Click to edit and rename your project.

![Project Name](docs/manual-screenshots/project_name_field_1774480063079.png)

---

#### Zoom Out (−)

Decreases the canvas zoom level, allowing you to see more of the slide at a smaller size.

![Zoom Out](docs/manual-screenshots/zoom_out_button_1774480071254.png)

---

#### Zoom In (+)

Increases the canvas zoom level, allowing you to see fine details of your slide at a larger size.

![Zoom In](docs/manual-screenshots/zoom_in_button_1774480072266.png)

---

#### Present Button

Launches a full-screen preview of your presentation. Slides advance automatically based on their configured duration. Press **Escape** to exit the presentation.

![Present](docs/manual-screenshots/present_button_1774480073277.png)

---

#### Schedule Button

Opens the scheduling dialog where you can set a start time, end time, and recurrence pattern for this project to be displayed on connected screens.

![Schedule](docs/manual-screenshots/schedule_button_1774480074296.png)

---

#### Theme Toggle (☀/🌙)

Switches the editor interface between **Light Mode** and **Dark Mode**. This only affects the editor UI, not the slide content.

![Theme Toggle](docs/manual-screenshots/theme_toggle_button_1774480075317.png)

---

#### Save Button

Manually saves all changes to the current project. The button is highlighted in blue to draw attention. It's recommended to save frequently. Unsaved changes may be lost.

![Save](docs/manual-screenshots/save_button_1774480076326.png)

---

### 7.5 Properties Panel

Located on the right side of the editor. Displays contextual properties for the currently selected element or the slide itself.

---

#### Slide Duration Slider

Controls how long (in seconds) the current slide is displayed during a presentation. Drag the slider or click to adjust. The value is shown on the right side (e.g., "10s").

![Duration Slider](docs/manual-screenshots/duration_slider_1774480084186.png)

---

#### Background Color Picker

Sets the background color of the current slide. Click the color swatch to open a color picker, or type a hex code directly.

![Color Picker](docs/manual-screenshots/color_picker_1774480085202.png)

---

#### Brand Color Presets

Three quick-select color circles below the color picker for applying your organization's brand colors with a single click:

- **Stratus Blue** — The primary brand color
- **Stratus Gray** — A neutral secondary color
- **Dark Slate** — A dark accent color

![Brand Color Presets](docs/manual-screenshots/brand_color_1_1774480086212.png)
